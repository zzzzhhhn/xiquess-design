# Teleport 节点传送

可以将被包裹的dom节点传送到指定dom节点下。

### 基本用法

```html
<template>
    <Teleport to="#app">
        <div>template<div>
    </Teleport>
</template>

```

### Attributes

| 参数              | 说明                             | 类型            | 可选值 | 默认值 |
| ----------------- | -------------------------------- | --------------- | ------ | ------ |
| to              | 目标节点   | string          |        |        |
