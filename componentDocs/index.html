# 简介

喜鹊DesignUI是由智邦国际·喜鹊前端团队开发并维护的开源vue组件库，包换喜鹊项目开发过程中因为常用组件库（element-ui、iview）无法满足需求而自主开发的一些特殊组件，本着互联网的分享精神，免费开源供大家使用。
